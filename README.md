# Electric Vehicle (EV) Analysis Dashboard – Tableau

## 📌 Overview

This project presents an interactive Tableau dashboard analyzing Electric Vehicle (EV) adoption trends across Washington State.

The dashboard explores:

* EV adoption by vehicle type
* Top EV manufacturers
* Geographic distribution of EV ownership
* Vehicle year trends
* County-level EV penetration

---

## 🔧 Tools & Technologies

* Tableau
* SQL
* Data Visualization
* Dashboard Analytics
* Geospatial Mapping

---

## 📊 Dashboard Features

* Interactive filters and slicers
* Manufacturer comparison analysis
* EV trend analysis by year
* County-level geospatial mapping
* Dynamic user-driven exploration

---

## 📈 Key Insights

* Tesla dominates EV registrations across Washington State.
* Battery Electric Vehicles (BEVs) account for the majority of EV adoption.
* EV adoption accelerated significantly after 2021.
* King County recorded the highest EV concentration.
